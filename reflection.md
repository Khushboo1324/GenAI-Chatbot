# Reflection

## Architecture and Choices
- Built with **Next.js (App Router)** for simple routing and server-side API capabilities.
- Styled using **Tailwind CSS** for a responsive, clean, and modern interface.
- Used **React State** and **React Hooks** (`useState`, `useRef`, `useEffect`) to manage conversation state, persona switching, typing indicators, and auto-scrolling.
- Integrated **OpenAI API** in a secure backend route to keep the API key hidden from the client.
